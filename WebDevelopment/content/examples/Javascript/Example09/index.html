Leeruitkomsten

Je kunt op basis van organisatie requirements (Analyseren) een vertaling maken naar een ontwerp (Toepassen) voor een full stack webapplicatie. Volgens dit ontwerp bouw (Creëren) je vervolgens met moderne technieken een web applicatie door gebruik te maken van frameworks. Je maakt gebruik van een database voor de opslag van gegevens.

Competenties

Voor alle competenties geldt dat ze gericht zijn op het kunnen maken van een professionele full stack web applicatie.
Onderstaande architectuurlagen zijn van toepassing. Deze moeten op het juiste niveau worden aangetoond:

  • Software : niveau 2
  • Gebruikersinteractie : niveau 1
  • Professioneel Handelen : niveau 1

Section 4

De volgende vakken worden gegeven:

  • Object Oriented Design & Programming
    Er is aandacht voor het ontwerpen van klassendiagrammen en het toepassen van OOP-concepten zoals klassen, methoden, overerving, overloading en dergelijke
  • Programmeren met Model-View-Controller & REST-API
    Met dit vak wordt de student stap voor stap geleerd hoe een overstap gemaakt kan worden van een traditionele web applicatie naar een moderne architectuur met MVC. Hiervoor wordt een keuze gemaakt in een modern framework als Angular, React of Microsoft .NET Core.
  • Database ontwerp en theorie
    Dit vak bouwt verder op bestaande basis kennis van database gebruik en gaat dieper in op hoe een database ontworpen en gebruikt kan worden. De student leert hoe een database met SQL-statements aangemaakt en bevraagd kan worden
  • ORM
    Ontsluiting van de database via een ORM in een applicatie. Het ORM wordt geïmplementeerd met een bestaand framework zoals Microsoft Entity Framework.
  • Functioneel ontwerp
    De student leert hoe een basisontwerp geschreven kan worden. Omdat tegenwoordig steeds vaker in Agile teams (bijv. scrum) wordt gewerkt is er aandacht voor Agile werken (ontwerpen, prioriteren, scrum board)
  • Front-end development & interactie design
    De student leert wat er nodig is om een aantrekkelijke gebruikersinterface te ontwerpen en implementeren.
